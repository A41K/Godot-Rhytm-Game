On the Dot (OTD) is a simple rhythm game developed in the GODOT engine

Features:
- Up/Down note spawning
- Complete charting system
- Arrow types
- Controller playabality
- Story Mode
- Freeplay
- 2 Minigames (Currently)


**DISCLOSE**

Ai was used for debugging and idea generation. Debugging for the Platformer minigame, Note spawning.
The ideas were for implementation and they were heavily altered/changed then what it was. I wanted to disclose
this so that everyone knows. Ai is for ideas only and debugging (in my opinion). Early charting was made with AI 
aswell to speed up the process and to test different features
